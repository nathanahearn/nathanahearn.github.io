D365 & Power Automate

D365

In my time with Dynamics 365, I have have performed different roles and tasks for a number of clients. I have been a part of implementations and projects with tasks ranging from data migration, specific customization, end user training and system documentation. As a Functional Consultant, I have seen many different components of Dynamics which inspired me to get my Customer Service, Sales, Marketing & Power Platform certifications.

Power Automate

Here are some elements of Power Automate and how I used them to provide solutions:

  • Environment Variables
    1. Ability to run automations in Production as well as in Sandbox environments.
  • Web Hooks
    1. In one example I've used this to fetch information from an Document Location in SharePoint.
  • FetchXML and ParseJSON
    1. For FetchXML it provides another way to filter the list of records retrieved by the List Records function in Power Automate.
    2. For ParseJSON it allows access to the text retrieved from different fields that do not always show up in Dynamic Content that still provide value.
